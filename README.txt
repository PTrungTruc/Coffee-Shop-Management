--- Requirement ---
* Visual Studio (install all neccessary to run ASP.NET project)
* SQL Server Management Studio Management Studio

--- How to run the project ---
1) Extract the project file zip to path you want.
2) Open the project folder and run the 'jojo.sln' file.
3) Open 'SQL Server Management Studio Management Studio' and connect to './SQLEXPRESS'.
4) Create a 'jojdb' database and run the 'jojodb.sql' which is in the project folder.
5) Build the project and run it in VS.
